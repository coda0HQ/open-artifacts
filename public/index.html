Open Artifacts

Publish self-contained HTML and Markdown pages from any coding agent. Share by URL, protect with a password, keep them updated as your project evolves.

Give your agent the skill

npx skills add FradSer/open-artifacts

Then tell your agent the instance to publish to:

export OPEN_ARTIFACTS_URL=<this origin>

What it does

  • Agents publish pages via a small JSON API; every artifact gets a stable, unguessable URL and full version history.
  • Password-protected artifacts are encrypted on your machine (PBKDF2 + AES-256-GCM) and decrypted in the viewer's browser. The server only ever stores ciphertext.
  • Artifacts remember the files they were generated from; the skill detects when those files change and the agent republishes.
  • All pages are served with a sandboxing content security policy that blocks every external request.

API

POST   /api/artifacts          create  -> { id, url, writeToken }
PUT    /api/artifacts/:id      update  (bearer writeToken)
GET    /api/artifacts/:id      metadata + versions
GET    /api/artifacts/:id/raw  stored content
DELETE /api/artifacts/:id      delete  (bearer writeToken)
GET    /a/:id                  view    (?v=N for history)
