Validation gates

Paste an HTML body snippet. The gates below light up as they trip — mirroring the real local builder.

Try adding <style>, a remote <img src="https://">, or two style= attributes on one tag.

  • •body <style>/<script>HTML body fragments cannot contain <style> or <script>.
  • •external requestsCDN scripts, web fonts, remote <img>, fetch/XHR/WebSocket all fail the CSP.
  • •dup style=A start tag carrying style= twice fails — parsers keep only the first.
  • •contrast P0Authored --fg/--bg/--accent overrides must clear 4.5:1 in both themes.
  • •container-classA CSS .class with max-width must be applied in the body.
  • •measure cap (L1)An L1 non-canvas HTML doc must constrain body width.
  • •canvas ABIOne #canvas + one #plane; frames as <section> with kebab ids and geometry.
  • •frame overlapTwo canvas frames whose interiors intersect fail.
